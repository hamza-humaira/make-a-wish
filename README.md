# 🌌 Make A Wish

### *A Shared Digital Sky for Anonymous Wishes*

**Make A Wish** is an atmospheric web experience where users can draw their own stars, attach personal wishes to them, and release them into a shared night sky.
Each glowing star in the universe contains a hidden anonymous thought, dream, memory, or hope written by someone else somewhere in the world.
Rather than feeling like a traditional social media platform, the project is designed to feel **calm, intimate, emotional, and immersive** — like a quiet moment beneath a shared sky.

# ✨ Core Concept

The application transforms anonymous wishes into floating celestial objects inside a digital universe.

Users can:

* ⭐ Draw their own custom star
* ✍️ Write a wish anonymously
* 🌠 Launch the wish into the sky
* 🔭 Explore wishes written by strangers
* 🌌 Experience a calming digital universe

The experience focuses heavily on:

* **Atmosphere**
* **Emotion**
* **Visual immersion**
* **Wonder**
* **Quiet connection between strangers**

# ✨ Features

## ⭐ Star Creation

Users can create and personalize stars before launching them into the universe.

## ✍️ Anonymous Wishes

Every star contains a private anonymous message written by a user somewhere in the world.

## 🌌 Shared Sky

All users contribute to the same digital universe, creating a constantly evolving sky of wishes.

## ✨ Atmospheric Animations

Twinkling stars, glowing effects, drifting motion, and smooth transitions create a calming experience.

## 📱 Responsive Design

Optimized for both desktop and mobile devices.

## 🔊 Ambient Experience

Soft visuals and motion are designed to create emotional immersion rather than fast-paced interaction.


# 🧠 Product Vision

Most online platforms are designed to maximize attention.

**Make A Wish** is designed to create *stillness*.

The goal is not endless engagement.
The goal is emotional presence.

The project explores the idea that strangers around the world may share similar hopes, fears, memories, and dreams — represented together as stars in a shared universe.
